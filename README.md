# E-commerce Sales & Product Analysis (SQL Project)

## Project Overview
This project analyzes customer behavior, product performance, sales trends, and return patterns using SQL.

## Dataset Tables
- boat_customer
- order_boat
- order_detail
- products
- returns

## Key Insights
- Identified top 5 customers by revenue
- Found most sold and most returned products
- Calculated return rate and delivery time
- Analyzed monthly sales trends

## Tools Used
- MySQL
- SQL (Joins, Aggregations, Date Functions)
